# AtomicMindustry
Mindustry MOD
このMODは元素とその化合物を中心に追加していきます。
前提MODとして使用はできます。
(もし良ければどのように使ってるか教えてください。)
また、追加して欲しいものは追加します。

This mod will focus on adding elements and their compounds.
You can use it as a prerequisite mod.
